# zoo
Database with animals. It allows you to display information about the species and a specific animal.


## Technologies 
* Java 1.8
* Baza danych: SQLlite

### Executing programs
Startup instruction.

* Open the zoo/src/main/java/pl.sytomczak.zoo/gui/MainVIew/MenuView module.
* Run the `MenuView` class.
